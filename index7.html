Inline vs Block Elements

This is the block element. This one is the inline. I am using a span element. I am using a paragraph element for the block element.

What are Inline and Block elements?

Block

Start on a new line and are the full width of the container. Like words in a paragraph.

Inline

Only as wide as needed and stays on the same line. Like words in a sentance.

Inline Block

Inline but set size and width and height apply. Like buttons in a row.
